자바스크립트 자료형

변수variable와 상수constant

변수는 데이터를 담는 그릇 - var, let으로 선언

상수는 변하지 않는 값을 가진 그릇 - const으로 선언

변수/상수 선언시 자료형을 기술할 필요 x

변수 선언시 var 이나 let (ES6에서 추가)을 사용

실무에서는 오류를 최소화하기 위해 LET(중복 불가)의 사용을 권장함

자료형

자바스크립트 자료형에는 Number,String,Boolean,Null,Undefined,BigInt,객체형등이 있음

숫자형

Number형으로 정수 , 실수등을 다룰 수 있음

BigInt형은 기존 정수보다 더 많은 범위의 수를 다룰 수 있음

한편, Number형에는 일반적인 숫자외에 추가적으로 NaN(not a number), Infinity/-Infinity등의 특수 숫자값 사용할 수 있음

따라서, 말도 안되는 연산에도 오류는 발생하지 않음

문자형

단일 문자나 단일문자가 여러개 모인 문자열을 다룰 때 사용

문자열을 정의 할 때는 '',"",``등을 이용함

Boolean형

논리형의 일종으로 true,false등의 논리값만 존재

주로 긍정(true),부정(false)을 나타내는 값에 사용

Null

null 값만 다루는 자료형

null 은 존재하지 않는 값, 비어 있는 값 , 알수 없는 값을 나타내는데 사용

undefined

undefined값만 다루는 자료형

undefined는 변수를 선언했지만, 값이 할당 되지 않은 상태를 나타내는데 사용

객체형

String형 , Number형 , Boolean형은 변수에 단 하나의 값만 표현가능 - 원시 primitive자료형

하지만 , 복잡한 프로그램의 경우 이것으로는 다소 부족

데이터의 집합이나 개체를 표현하기 위해 객체 자료형이 도입 - 변수 하나에 여러 종류의 데이터들을 담을 수 있음
