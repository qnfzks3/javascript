자바스크립트 객체

개요

자바스크립트는 기본 자료형을 제외한 나머지는 모두 객체임

자바스크립트는 기본적으로 key 와 value 로 구성된 속성들의 집합

자바스크립트의 함수는 1급 시민이므로 값으로 취급 가능

따라서 , 객체는 데이터를 의미하는 속성과 데이터를 참조하고 조작하는 동작을 의미하는 메서드(선언된 함수)로 구성할 수 있음

속성 property

키와 값으로 구성된 객체의 주된 구성요소

객체 속성 접근하기

객체 속성에 접근하거나 다룰려면 '객체명.키 이름'형식 사용

또는 '객체명[키이름]'으로도 사용 가능

객체 내용 순회

for - in문을 사용하면 객체에 포함된 모든 속성에 접근할 수 있음

다양한 형태의 객체

객체의 요소로 객체 또는 함수를 포함 시킬 수도 있음

Set 객체

배열과 유사한 자료구조

하지만, 배열은 저장순서를 기억하고 , 중복값 허용하지만 , Set은 중복값은 허용하지 않음

Map

키와 값으로 구성하는 Object와 유사한 자료구조

하지만, Object는 속성과 함수를 포함할 수 있지만, Map은 속성만 포함할 수 있음

난수 생성 v1

특정 범위에 속하는 정수 중 임의의 숫자를 뽑으려면 다음 함수와 수식을 이용함

1. Math 객체의 random 함수 사용: 1~0 사이 부동 소수점 의사 난수 생성

2. 난수 생성식 : random % 범위최대값 +1

2. 난수 생성식 : random * 범위최대값 +1 (최대 값 지정 )ex) *45로 하면 범위는 0~44 총 45개
